# subsonic-github-readme

SVG template from https://github.com/natemoo-re/natemoo-re

## Usage
```python
pipenv run python3 main.py
```

## github actions
Populate repo secrets
```env
USERNAME=
TOKEN= # md5(${PASSWORD}${SALT})
SALT= # random string
BASE_URL=https://EXAMPLE.COM # no trailing slash
```
